The following environment variables will be needed for the API
to run:

	Environment Variables:
	AZURE_STORAGE_ACCOUNT: name of Azure Storage account
	AZURE_STORAGE_ACCESS_KEY: private key for Azure Storage

	OAUTH_KEY: identifier for myFields.info authentication
	OATH_SECRET: private key for myFields.info authentication

After installing all dependencies, the API can be started using
'npm start' with elevated privileges.
